# Getting Your Apify API Token

## Step 1 — Create an Apify account
Go to [apify.com](https://apify.com) and sign up for a free account.

## Step 2 — Open your account settings
After logging in, click your profile icon (top-right) and select **Settings**.

## Step 3 — Navigate to the API & Integrations tab
In the left sidebar, click **API & Integrations**.

## Step 4 — Copy your token
Under **Personal API tokens**, your default token is shown. Click **Copy** to copy it to your clipboard.

> You can also create a new token here if you want a dedicated one for this project.

## Step 5 — Paste it into your `.env`
```
APIFY_API_TOKEN=your_token_here
```
